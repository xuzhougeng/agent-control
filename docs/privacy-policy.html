Privacy Policy — CC Remote

Last updated: February 2025

CC Remote ("the App") is an iOS client for connecting to your own cc-control server. This policy describes what data the App uses and where it stays.

Data the App Uses

  • Server URL — The base URL of your cc-control server that you enter in Settings. It is stored only on your device (iOS UserDefaults).
  • UI Token — The authentication token you enter in Settings. It is stored only in your device's Keychain and is used only when communicating with the server URL you configured.
  • Terminal and session data — All terminal input/output and session state are exchanged only between the App and your cc-control server. We do not have access to this data.

What We Do Not Do

  • We do not collect, store, or transmit your data to any third party or to our own servers.
  • The App does not include analytics, advertising, or other SDKs that collect personal data.
  • We do not have access to your server URL, token, or any content you type in the terminal.

Your Server

You are responsible for the privacy and security of your cc-control server and any data processed there. The App only connects to the server you configure; it does not send data elsewhere.

Changes

We may update this policy from time to time. The "Last updated" date at the top will be revised when we do. Continued use of the App after changes constitutes acceptance of the updated policy.

Contact

For questions about this policy or the App, use the support link provided in the App Store listing (e.g. the project repository or documentation site).
